# Pilot Quick Start (Teachers)

## Goal
Run one focused intervention and generate one meeting-ready summary in under 10 minutes.

## 1) Open the teacher workflow
- Go to `teacher-hub-v2.html` for the daily operating surface.
- Select a block or student who needs support now.

## 2) Review the snapshot (30-60 seconds)
- Read the morning brief and selected block context.
- Confirm the tier/support line and recommended move.

## 3) Launch one action
- Click **Open Lesson Brief** or launch the recommended activity.
- Complete one short instructional run.

## 4) Log and return
- Return to the Hub or open the student profile if you need one-student detail.
- Confirm the student/context remains selected.

## 5) Generate meeting output
- Open `reports.html` for the prep workspace.
- Click **Meeting Prep** or **Weekly Insight**.
- Use Summary/Deck/Notes/Export tabs.
- Set language if needed.
